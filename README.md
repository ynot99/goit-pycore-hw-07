## Commands example for the assistant bot

```bash
hello

all

birthdays

phone John

change John 1234567890 0987654321

show-birthday John

add-birthday John 8.11.2025

add John 1234567890

close

exit
```

# Project structure

```
core/: For now only contains core exceptions for the program
contacts/: Contains contact-related classes and fields
contacts_assistant/: Main program logic for taking user input and managing contacts
```


## Setup

### Create and activate a virtual environment

```bash
python3 -m venv venv
```

```bash
source venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Command to run the code

```bash
python3 module10_task1and2.py
```
